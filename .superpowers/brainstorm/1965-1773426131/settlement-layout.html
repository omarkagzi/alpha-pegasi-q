Arboria Market Town — Settlement Layout

This is the first (and only MVP) walkable settlement. How should it be structured?

A: Central Market Square

Classic layout. Open market square at center, agent homes radiating outward. Park area to the south. Paths connect all areas. Player spawns at north entrance. Simple, readable, easy to navigate.

┌──────────────────────────────────┐ │ RESIDENTIAL │ MARKET │ │ 🏠 Agent1 │ ┌────────┐ │ │ 🏠 Agent2 │ │ Stalls │ │ │ 🌳 Garden │ │ Plaza │ │ │────────────────┼──┘ │ │ │ COMMONS │ └────────┘ │ │ 🌳 Park 🌳 │ │ │ ⛲ Fountain │ CRAFT ROW │ │ 🌳 Benches │ 🏠 Agent3 │ │ │ 🏠 Workshop │ └──────────────────────────────────┘

B: Distinct Districts

Town divided into named districts: Residential, Market, Commons (park), Craft Row. Each area has its own character. More complex to navigate but feels like a real town. Supports future expansion — add new districts as agents register.

┌──────────────────────────────────┐ │ 🌳🌳 GATE 🌳🌳 │ │ ║ │ │ 🏠 ═══╬═══ 🏠 │ │ Agent1 ║ Agent2 │ │ ║ │ │ 🏠 ═══╬═══ 🏠 │ │ Shop ║ Agent3 │ │ ║ │ │ 🌳 ═══╬═══ 🌳 │ │ Park ║ Tavern │ │ ║ │ │ 🌳🌳 SQUARE 🌳🌳 │ └──────────────────────────────────┘

C: Main Street Linear

One main road from gate to square, with buildings on both sides. Most linear and guided experience — hard to get lost. Very Stardew Valley. But less room for exploration and feels smaller.

Note: The MVP needs exactly 3 demo agent homes + ambient buildings (shop, tavern, etc.) + park/green space. The map will be authored in Tiled and exported as .tmj for Phaser 3. Approximate target size: 40×30 tiles at 16px or 32px per tile.
