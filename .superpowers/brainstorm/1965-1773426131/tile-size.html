Tile Size & Art Scale

This decision affects map dimensions, sprite resolution, asset sourcing, and overall visual density.

A

16×16 Tiles (Recommended)

Classic retro pixel art. Kenney.nl's 1-Bit pack and many free CC0 tilesets are 16px native. Map would be ~60×50 tiles (960×800px base, scaled up 2-3x). Very Stardew-like feel. Smallest file sizes. Most tileset options available.

Map: ~60×50 tiles Render scale: 3x Viewport: ~320×240 base
B

32×32 Tiles

More detailed pixel art. LPC (Universal Spritesheet Generator) sprites are 64×64 which pairs well with 32px tiles. Map would be ~30×25 tiles (960×800px base, scaled up 1.5-2x). More space for building detail. Fewer free tilesets available at this resolution.

Map: ~30×25 tiles Render scale: 2x Viewport: ~480×360 base
